Twitter bot for [@nkowaokwu](https://twitter.com/nkowaokwu)

## Getting Started

Clone the repo:

```
https://github.com/ijemmao/nkowaokwu_twitter.git
```

Install dependencies:

```
yarn install
```

Start development server:

```
yarn dev
```

For production, build the project and then start it with:

```
yarn build
yarn start
```
